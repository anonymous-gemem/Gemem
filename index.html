Interactive World Generation with
Explicit 3D Memory
Inspired by the way humans picture, store, and mentally replay a 3D world they’ve been in, we present Gemem: a generative world model that pairs interactive video generation with the explicitly reconstructed 3D memory. Starting from a single panoramic image, \ours offers fine-grained view control to generate future video frames while dynamically reconstructing the 3D memory from previously generated frames. Crucially, it projects this evolving 3D reconstruction onto target views as conditional inputs to a video generation model, thereby providing explicit spatial cues that enhance both realism and consistency. To evaluate our framework, we curate the first thorough dataset for long-range and loop-closure exploration across synthetic outdoor, Habitat indoor, and real-world scenes. Extensive experiments confirm that the explicitly 3D memory significantly improves visual fidelity and spatial scene coherence, offering a promising step toward long-horizontal spatially consistent world modeling.
